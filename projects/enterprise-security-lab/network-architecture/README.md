# 🌐 Network Architecture

## Overview

This section documents the enterprise network design and security architecture implemented in the lab environment.

The environment simulates a small corporate network with segmentation, firewall protection, and security monitoring.

## Network Components

- OPNsense Firewall
- VLAN Segmentation
- Active Directory Domain
- Windows Server
- Linux Servers
- SOC Monitoring Network

## Network Segmentation

- VLAN 10 - Management Network
- VLAN 20 - Corporate Network
- VLAN 30 - DMZ Network
- VLAN 40 - SOC Network

# Cyber Defence Labs

This folder contains all the labs for the **Network Defense / Cybersecurity** course.  
The purpose of these labs is to provide hands-on experience in configuring firewalls, network security, NAT, and port forwarding using Linux and iptables.

---

## Contents

| Lab | Brief Description | Link |
|-----|-----------------|------|
| Lab 1: iptables Configuration | Full iptables firewall configuration including basic rules, IP/port filtering, stateful firewall, MAC filtering, and NAT | [Lab 1](Lab1-iptables/) |
| Lab 2: Linux System Hardening | Implemented Linux system hardening techniques including kernel management, SSH configuration, firewall setup, and security updates | [Lab 2](Lab2-linux_sys_hardening/) | (./Linux-Hardening-Lab-Report.pdf)

## How to Use

1. Navigate to each lab folder.  
2. Each lab folder contains:
   - **README.md** with detailed instructions and commands.
   - `screenshots/` folder with screenshots (if available).
   - `resources/` folder containing configuration files and backups.
3. Click on the link in the table above to access the full lab instructions.

---

## Naming Conventions

- Avoid spaces in folder and file names; use `-` or `_` instead.  
- Example: use `Lab1-iptables` instead of `Lab 1 iptables`.  
- GitHub automatically displays `README.md` in each lab folder.
